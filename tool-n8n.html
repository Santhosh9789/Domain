n8n

Workflow automation for technical people.

Why n8n?

n8n is an extendable workflow automation tool. It allows you to connect anything to everything via its open, fair-code model. It's particularly powerful because it can be self-hosted, giving you complete data control.

  • Node-Based: Visual workflow editor with hundreds of native integrations.
  • Self-Hosted: Keep your data secure on your own servers, or use their cloud version.
  • Customizable: If a node doesn't exist, you can write your own JavaScript or build a custom node.
