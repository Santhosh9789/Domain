Automate browsers. That's it!

Why Selenium?

Selenium is an umbrella project for a range of tools and libraries that enable web browser automation. It is the core technology used in countless browser automation tools, APIs, and frameworks.

  • Testing Automation: Create robust, browser-based regression automation suites and tests.
  • Cross-Browser: Run tests against Firefox, Chrome, Edge, Safari, and more on Windows, Mac, and Linux.
  • Language Support: Write tests in Java, Python, C#, Ruby, JavaScript, and Kotlin.
