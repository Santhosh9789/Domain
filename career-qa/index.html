QA Tester

Freelance | 0-2 Years Experience

Ensure Quality Excellence

We are seeking a detail-oriented QA Tester to test our web and mobile applications. You will identify bugs, inconsistencies, and usability issues to ensure a flawless user experience.

Key Responsibilities

  • Manual Testing: Execute test cases for web and mobile applications to identify functional and UI defects.
  • Bug Reporting: Document bugs with clear reproduction steps and track them to closure.
  • Compatibility: Ensure applications work correctly across different browsers and devices.

Requirements

  • Experience: 0-2 years in software quality assurance or testing.
  • Knowledge: Understanding of SDLC and STLC concepts.
  • Attention to Detail: Keen eye for design discrepancies and functional errors.
  • Tools: Familiarity with bug tracking tools like Jira or Trello is a plus.

Compensation

  • Payment Model: Per Project Milestone.
  • Flexibility: Project-based engagement.
  • Remote Work: 100% remote.
