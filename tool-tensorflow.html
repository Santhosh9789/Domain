TensorFlow

An end-to-end open source machine learning platform.

Why TensorFlow?

TensorFlow has a comprehensive, flexible ecosystem of tools, libraries and community resources that lets researchers push the state-of-the-art in ML and developers easily build and deploy ML powered applications.

  • Easy Model Building: Build and train ML models easily using intuitive high-level APIs like Keras.
  • Robust Production: Run your models anywhere, from cloud, on-prem, in the browser, or on-device.
  • Powerful Experimentation: A simple and flexible architecture to take your new ideas from concept to code to state-of-the-art models.
