Cloud & DevOps Engineer

Remote | 3-6 Years Experience

Build Robust Infrastructure & Pipelines

We are looking for a skilled DevOps Engineer to manage our cloud infrastructure and deployment pipelines. You will ensure our systems are reliable, scalable, and secure.

Key Responsibilities

  • Infrastructure Management: Design, deploy, and manage cloud infrastructure using AWS or Azure.
  • CI/CD Pipelines: Build and maintain automated CI/CD pipelines (Jenkins, GitHub Actions, GitLab CI) for seamless deployment.
  • Containerization: Expertise in Docker and Kubernetes for container orchestration and management.
  • Monitoring & Security: Implement monitoring tools (Prometheus, Grafana) and ensure robust security practices across infrastructure.

Requirements

  • Experience: 3+ years experience in a DevOps or Cloud Engineering role.
  • Cloud: Strong hands-on experience with AWS (EC2, S3, RDS, Lambda) or Azure.
  • scripting: Proficiency in scripting languages like Python, Bash, or Go.
  • IaC: Experience with Infrastructure as Code (Terraform, CloudFormation).

Compensation

  • Project-Based: Competitive fees for setup, migration, and maintenance projects.
  • Retainer: Monthly retainer options for ongoing infrastructure management.
  • Remote Work: 100% remote.
