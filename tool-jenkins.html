Why Jenkins?

Jenkins is the leading open source automation server. Built with Java, it provides hundreds of plugins to support building, deploying, and automating any project.

  • Continuous Integration/Delivery: As an extensible automation server, Jenkins can be used as a simple CI server or turned into the continuous delivery hub for any project.
  • Plugins: With hundreds of plugins in the Update Center, Jenkins integrates with practically every tool in the continuous integration and continuous delivery toolchain.
  • Distributed: Jenkins can easily distribute work across multiple machines, helping drive builds, tests and deployments across multiple platforms faster.
